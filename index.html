我放在 github 上的個人專案

您可以找到這些東西：

iPhone 程式

  • TW Weather－台灣天氣
    • 這個東西當然是拿來看天氣資訊的，不然還能夠有什麼用途？
    • 包含 iPhone client 端與 App Engine server 端程式
    • iPhone 使用需 iPhone OS 3.0 以上版本
    • 自行編譯需要 Mac OS X 10.5 以上作業系統之 Intel Mac，以及 iPhone SDK 3.0 以上版本

Mac OS X 程式

  • IMRemoval－Mac OS X 輸入法移除工具
    • 幫你刪除之前安裝過的第三方輸入法
    • 可在 Mac OS X 10.4 以上版本使用
    • 自行編譯需要 Mac OS X 10.5 以上作業系統之麥金塔電腦，以及 Xcode 3.0 以上版本
  • tcfail－Mac OS X 10.6 繁體中文預設字體修改工具
    • 幫你把預設的中文字體換成別的字體
    • 需在 Mac OS X 10.6 以上版本使用
    • 自行編譯需要 Mac OS X 10.6 以上作業系統之麥金塔電腦，以及 Xcode 3.2 以上版本
  • bunt－色彩相關程式碼產生工具
    • 覺得自己轉換產生 NSColor、UIColor 時需要的參數很煩嗎？
    • 可在 Mac OS X 10.5 以上版本使用
    • 自行編譯需要 Mac OS X 10.5 以上作業系統之麥金塔電腦，以及 Xcode 3.0 以上版本
  • ASCII Converter
    • 把圖片轉成 ASCII Art 的工具，純粹就是個好玩的工具
    • 可在 Mac OS X 10.5 以上版本使用
    • 自行編譯需要 Mac OS X 10.5 以上作業系統之麥金塔電腦，以及 Xcode 3.0 以上版本

Web Apps

  • KeyKeyTools－輸入法工具箱
    • 提供像是將新注音輸入法自定詞庫，轉換成 Yahoo! 奇摩輸入法或自然輸入法格式等工具，直接使用網頁操作。
    • URL: http://keykeytools.zonble.net/
